# 🤖 Customer Feedback AI Agent

A smart customer feedback automation workflow built using **n8n** and **Google Gemini AI**. The system automatically analyzes customer feedback, categorizes it, and notifies the appropriate team without any manual intervention.

## Features

- Accepts customer feedback through a simple form.
- Uses Google Gemini AI to classify feedback as **Complaint**, **Compliment**, or **Feature Request**.
- Automatically stores all feedback in Airtable for easy tracking and management.
- Sends real-time Slack notifications to keep the team informed.
- Sends email notifications through Gmail based on the feedback category.
- Eliminates manual sorting of customer feedback and speeds up the response process.
- Designed to improve customer support by automating repetitive tasks.

## Tech Stack

- n8n
- Google Gemini AI
- Airtable
- Slack
- Gmail
